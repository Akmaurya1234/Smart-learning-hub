Introduction to C++

C++ is a powerful general-purpose programming language. It supports procedural, object-oriented, and generic programming:

  • History: Developed by Bjarne Stroustrup as an extension of C.
  • Features: Supports OOP, memory management, and performance optimization.
  • Uses: Used in system/software development, game engines, drivers, and more.

Setup and Installation

To run C++ programs, you need a compiler and a code editor or IDE:

  • Compiler: Install g++ via MinGW (Windows) or terminal (Linux/macOS).
  • IDE: Use Code::Blocks, Visual Studio, or online platforms like Replit.
  • Run: Compile with g++ file.cpp -o output, then run the executable.

Syntax

C++ syntax is similar to C, but includes enhancements for object-oriented programming:

  • Semicolon: Each statement ends with ;
  • Braces: Code blocks are enclosed in { }
  • Main Function: Entry point of any C++ program is int main()

Variables & Data Types

Variables are used to store data, and each has a data type:

  • int: Stores integers, e.g., int x = 10;
  • float/double: For decimal values, e.g., float pi = 3.14;
  • char: Stores single characters, e.g., char grade = 'A';
  • string: For text (requires <string> header)

Operators

Operators perform operations on variables and values:

  • Arithmetic: +, -, *, /, %
  • Relational: ==, !=, >, <, >=, <=
  • Logical: &&, ||, !
  • Assignment: =, +=, -=, *=

Conditional Statements

Used to make decisions based on conditions:

  • if: Executes a block if condition is true
  • else: Executes if condition is false
  • else if: Checks multiple conditions
  • switch: Selects one of many blocks to execute

Loops

Loops allow repeating code blocks:

  • for: Loop with initialization, condition, and increment
  • while: Loop based on a condition
  • do-while: Executes block at least once

Functions

Functions are reusable blocks of code that perform specific tasks:

  • Declaration: int add(int, int);
  • Definition: Contains the actual code
  • Calling: add(5, 10);

Arrays in C++

An array in C++ is a collection of elements of the same data type stored in contiguous memory locations:

  • Declaration: int arr[5];
  • Initialization: int arr[] = {1, 2, 3};
  • Access: arr[0]

Pointers

Pointers are variables that store the memory address of another variable:

  • Declaration: int* ptr;
  • Address-of Operator: &x gives the address of x
  • Dereferencing: *ptr accesses the value at that address

Strings

C++ has two types of strings: C-style (char arrays) and string class from STL:

  • Declaration: string str = "Hello";
  • Concatenation: str1 + str2
  • Functions: length(), substr(), find()

Structures

Structures are user-defined data types that group variables of different types:

  • Declaration: struct Name { ... };
  • Usage: Person p;
  • Access: p.name = "John";

Object-Oriented Programming

OOP allows you to structure code using classes and objects:

  • Encapsulation: Hides internal state
  • Abstraction: Shows only essential details
  • Inheritance: Reuse existing classes
  • Polymorphism: Same interface, different implementation

Classes & Objects

Classes are blueprints for creating objects:

  • Class: Defined using class keyword
  • Object: Instance of a class
  • Access: obj.method() or obj.variable

Inheritance

Inheritance allows one class to acquire the properties of another:

  • Base Class: The original class
  • Derived Class: Inherits from base class using :
  • Access: Public, protected, private inheritance

Polymorphism

Polymorphism allows functions to behave differently based on context:

  • Compile-time: Function overloading, operator overloading
  • Run-time: Using virtual functions and inheritance
  • Override: Derived class redefines base class method

Encapsulation

Encapsulation binds data and methods that manipulate it within a class:

  • Private members: Hidden from outside access
  • Public methods: Used to access private data
  • Secure: Helps protect object integrity

Abstraction

Abstraction hides complex implementation and shows only essentials:

  • Abstract class: Contains at least one pure virtual function
  • Interface: Only method declarations, no body
  • Implementation: Provided by derived classes

Exception Handling

Exception handling is used to catch and manage runtime errors:

  • try: Wraps code that might cause an exception
  • catch: Handles exceptions
  • throw: Generates an exception

File Handling

C++ provides file stream classes to handle file input and output:

  • ofstream: Write to files
  • ifstream: Read from files
  • fstream: Read/write both
  • Functions: open(), close(), is_open()
