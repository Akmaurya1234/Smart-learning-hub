Introduction to C

C is a general-purpose, procedural programming language. It is known for its efficiency and control over system resources. Initially developed by Dennis Ritchie at AT&T Bell Labs in the 1970s, C is widely used for system programming, embedded systems, and applications requiring high performance.

  • C is compiled and provides low-level memory access.
  • It is the foundation for many modern programming languages like C++, Java, and Python.
  • Syntax is simple and straightforward, making it a good language for beginners.

Data Types in C

C has several built-in data types for storing different kinds of values. These data types include:

  • int: Used for integers, e.g., int num = 10;
  • float: Used for decimal numbers, e.g., float pi = 3.14;
  • char: Used for single characters, e.g., char letter = 'A';

  • char: Used for single characters, e.g., char letter = 'A';

Variables in C

A variable is a named storage location in memory used to hold a value. The value of a variable can be changed during the execution of the program.

  • Declaration: To declare a variable, specify its data type and name.
  • Initialization: You can also initialize a variable with a value at the time of declaration.
  • Example: int num = 10;

Operators in C

C provides a wide range of operators for performing operations on variables and values:

  • Arithmetic Operators: +, -, *, /, % for mathematical operations.
  • Relational Operators: ==, !=, <, >, <=, >= for comparisons.
  • Logical Operators: && (AND), || (OR), ! (NOT).
  • Assignment Operators: =, +=, -=, *=, /= for assigning values.

Control Statements

Control statements allow you to control the flow of execution in a program based on conditions:

  • if: Executes a block of code if a condition is true.
  • else: Executes an alternate block of code if the if condition is false.
  • else if: Adds more conditions to check multiple possibilities.

Loops in C

Loops are used to repeat a block of code multiple times:

  • for loop: Used when the number of iterations is known.
  • while loop: Used when the number of iterations is not known.
  • do-while loop: Executes the block at least once, and then repeats while the condition is true.

Functions in C

A function is a block of code that performs a specific task. It can be called from other parts of the program to execute the task:

  • Function Declaration: Specifies the function name, return type, and parameters.
  • Function Definition: The actual code of the function.
  • Function Call: Calls the function to execute its code.

Arrays in C

An array is a collection of variables of the same type stored at contiguous memory locations:

  • Declaration: An array can be declared with a specific size, e.g., int arr[5];
  • Initialization: Arrays can be initialized at the time of declaration, e.g., int arr[] = {1, 2, 3};
  • Access: Elements can be accessed using their index, e.g., arr[0].

Pointers in C

A pointer is a variable that stores the memory address of another variable:

  • Declaration: int *ptr; declares a pointer to an integer.
  • Dereferencing: *ptr accesses the value stored at the address.
  • Address-of Operator: & is used to get the address of a variable.

Structures in C

Structures are used to group variables of different data types together:

  • Declaration: A structure is declared using the struct keyword.
  • Accessing Members: Members of a structure can be accessed using the dot operator, e.g., person1.age.

File Handling in C

C provides functions to handle files. You can read from and write to files using functions like fopen(), fwrite(), and fclose().

  • fopen: Opens a file. Syntax: FILE *fopen('filename', 'mode');
  • fwrite: Writes data to a file.
  • fclose: Closes a file when done.
