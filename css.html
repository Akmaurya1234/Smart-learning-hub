CSS

CSS stands for Cascading Style Sheets. It is used to control the layout and appearance of HTML elements on a webpage. CSS brings life to the structure created by HTML by adding colors, spacing, fonts, and responsive design.

  • Stands for Cascading Style Sheets.
  • Controls layout, colors, fonts, spacing, and more.
  • Helps separate structure (HTML) from presentation (CSS).
  • Allows responsive design for different devices.
  • Supports animations, transitions, and custom styling.
  • Can be written inline, internally, or externally.

CSS Features:

  • Separates content from design for better organization.
  • Provides flexibility and control over layout.
  • Improves load time and maintenance.
  • Supports media queries for responsive design.
  • Enables styling with selectors and pseudo-classes.
  • Animation and transition capabilities.
  • Compatible across modern browsers.

What is a style sheet?

A style sheet is a set of rules used to describe the presentation of HTML content. These rules define how elements appear on screen, in print, or other media.

CSS Syntax and Usage

Basic CSS Concepts:

  1. Selectors: Patterns used to select HTML elements for styling.
  2. Properties: Define what aspect of the element to style (e.g., color, font-size).
  3. Values: Set the style for the selected element (e.g., red, 16px).
  4. Declarations: A combination of property and value (e.g., color: blue;).

Why Learn CSS?

  1. Enhance web pages visually.
  2. Create responsive, mobile-friendly designs.
  3. Improve user experience and branding.
  4. Essential skill for frontend development careers.

Basic CSS Syntax

CSS syntax consists of a selector followed by a declaration block. Each declaration includes a property and a value, separated by a colon (:) and ended with a semicolon (;). CSS helps you control the style and layout of web elements using these rules. 

selector {
          property: value;
          property: value;
          }

Types of CSS

1. Inline CSS

Styles are applied directly on the element using the style attribute.

<p style="color: red; font-size: 18px;">This is inline CSS</p>

2. Internal CSS

Styles are written inside a <style> tag within the <head> of an HTML document.

<style>
  p {
      color: blue;
      font-size: 16px;
  }
</style>

3. External CSS

Styles are placed in an external file with a .css extension and linked using the <link> tag.

<link rel="stylesheet" href="style.css">

CSS Syntax

CSS is written in a set of rules. Each rule consists of a selector and a declaration block.

selector {
  property: value;
  property: value;
}

Example:

p {
  color: green;
  font-size: 20px;
}

CSS Selectors

Element Selector

The element selector targets all HTML elements of a specific type. For example, p selects all paragraph tags. 

p {
  color: orange;
  font-size: 18px;
}

ID Selector

The ID selector uses the hash (#) symbol to style a single, unique element. 

#uniqueTitle {
  color: crimson;
  text-align: center;
}

Class Selector

The class selector uses a dot (.) before the class name. It can be reused across multiple elements. 

.highlight {
  background-color: yellow;
  padding: 10px;
}

Universal Selector

The universal selector (*) targets all elements in the document. 

* {
  margin: 0;
  padding: 0;
}

Group Selector

The group selector applies the same styles to multiple elements using commas. 

h1, h2, p {
  color: teal;
}

Descendant Selector

This selector targets all elements inside a specific parent. 

article p {
  color: purple;
}

Child Selector

Selects only **direct** children of an element using >. 

div > p {
  color: green;
}

Adjacent Sibling Selector

The adjacent sibling selector (+) selects an element that is directly after another specific element, at the same level. 

h1 + p {
  color: deeppink;
  font-style: italic;
}

General Sibling Selector

The general sibling selector (~) selects all elements that are siblings of a specified element and appear **after** it in the DOM. 

h1 ~ p {
  color: orange;
  font-weight: bold;
}

Attribute Selector

Attribute selectors let you target elements based on their attributes and values. They're perfect for form styling. 

input[type="text"] {
  background-color: #333;
  color: white;
  border: 2px solid lightblue;
}

[attribute=value]

Selects element with a specific attribute value.

a[target="_blank"] {
  color: red;
}

[attribute^=value]

Selects attribute value that starts with something.

img[src^="logo"] {
  border: 2px solid green;
}

[attribute$=value]

Selects attribute value that ends with something.

a[href$=".pdf"] {
  color: orange;
}

[attribute*=value]

Selects if attribute contains a value.

a[href*="google"] {
  text-decoration: underline;
}

::first-letter

Styles only the first letter of the text.

p::first-letter {
  font-size: 2em;
  color: red;
}

::first-line

Styles only the first line of text.

p::first-line {
  font-weight: bold;
}

::placeholder Selector

Used to style placeholder text inside input fields.

:read-write Selector

Targets input fields that the user can modify (not readonly).

input:read-write {
  background-color: black;

CSS Specificity

CSS Specificity determines which rule is applied when multiple rules target the same element. More specific selectors override less specific ones. 

p {
  color: green;        /* Element selector (1 point) */
}
.highlight {
  color: blue;         /* Class selector (10 points) */
}
#unique {
  color: red;          /* ID selector (100 points) */
}
.highlight#unique {
  color: orange;       /* Class + ID = 110 points */
}

In the above example, the text will be orange because .highlight#unique has the highest specificity among them.

CSS Box Model

The Box Model defines how space is handled around elements. Each element has:

  • Content: Actual text/image.
  • Padding: Space inside border.
  • Border: Surrounds the padding.
  • Margin: Outside space of box.
.box {
    width: 200px;
    padding: 20px;
    border: 5px solid #1e90ff;
    margin: 20px auto;
  }

To control total size easily, use: box-sizing: border-box; This includes padding and border inside the width.

CSS Display Property

display defines how an element behaves on the page:

  • block: Takes full width, starts on new line (like <div>)
  • inline: Stays in line (like <span>), no width/height
  • inline-block: Like inline but supports size
  • none: Completely hides the element
.block { display: block; }
    .inline { display: inline; }
    .inline-block { display: inline-block; }
    .none { display: none; }

CSS Position Property

position controls how an element is placed on the page:

  • static: Default position (normal flow)
  • relative: Moves relative to its original place
  • absolute: Moves relative to nearest positioned ancestor
  • fixed: Stays fixed in viewport (even on scroll)
  • sticky: Scrolls with page but sticks when needed
.relative-box {
      position: relative;
      top: 10px;
      left: 20px;
    }
    
    .absolute-box {
      position: absolute;
      top: 10px;
      right: 10px;
    }

CSS Flexbox

Flexbox (Flexible Box Layout) is used to arrange elements in a **row or column** efficiently. It makes aligning, spacing, and distributing items easier â€” even when their size is dynamic.

Main Concepts:

  • Flex Container: The parent element (with display: flex).
  • Flex Items: The children of the container. These respond to flex properties.

Main Axis & Cross Axis:

  • Main Axis: Direction of items â€” default is left to right (row).
  • Cross Axis: Perpendicular to main axis â€” top to bottom (column).

Key Properties:

  • display: flex; Turns the container into a flexbox.
  • flex-direction: Row (default), row-reverse, column, column-reverse.
  • justify-content: Aligns items on the main axis (start, center, space-between...)
  • align-items: Aligns items on the cross axis (center, stretch, flex-start...)
  • flex-wrap: Allows items to wrap to the next line if needed.
.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
  }
  .box {
    background: #1e90ff;
    padding: 20px;
    margin: 10px;
    color: white;
  }

Bonus: You can use gap to create spacing between items easily!

CSS Grid Layout is a two-dimensional system for designing web layouts. It allows you to place items in rows and columns, giving you complete control over your layout structure. Itâ€™s perfect when you want a structured layout like a gallery, dashboard, or multi-column design.

ðŸ›  How it works:

  • display: grid; â€“ Converts a container into a grid layout.
  • grid-template-columns: â€“ Defines the number and size of columns.
  • grid-template-rows: â€“ (Optional) Defines the height of rows.
  • gap: â€“ Adds spacing between items (row and column).
.grid-container {
      display: grid;
      grid-template-columns: repeat(3, 1fr);
      gap: 10px;
    }

Explanation: - The container becomes a grid using display: grid. - repeat(3, 1fr) means "make 3 equal columns". - 1fr is a flexible unit: each column takes equal share. - The gap adds spacing between each grid item.

Output: The items align into a 3-column structure. More items will wrap onto new rows automatically.

CSS Transitions allow you to smoothly animate property changes on hover, focus, active, etc. This makes interactions feel more natural and professional.
