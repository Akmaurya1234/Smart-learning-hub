Basic HTML Concepts:

  1. Tags: HTML uses tags (<>) to define elements.
  2. Elements: Represented by tags, elements are the building blocks of HTML(e.g., headings, paragraph, images).

Document Structure Tags

1. <!DOCTYPE> Declaration

The <!DOCTYPE> declaration defines the document type and version of HTML. It ensures proper rendering across browsers. In HTML5, it is written as <!DOCTYPE html>.

2. <html> Root Element

The <html> tag is the root of every HTML document. It encloses all other elements and typically includes the lang attribute to specify the document language.

3. <head> Section

The <head> tag contains meta-information about the document, such as the title, character encoding, styles, and scripts.

4. <title> Tag

The <title> tag defines the webpage title, which appears in the browser tab and helps with SEO.

5. <body> Tag

The <body> tag holds all the visible content of the webpage, including text, images, and interactive elements.

What is an HTML Element?

An HTML element is a fundamental part of a webpage. It consists of a start tag, some content, and an end tag.

1. Block-Level Elements

Block-level elements always start on a new line and span the full width available. Common examples include:

  • <div> â€“ Defines a division or section
  • <p> â€“ Represents a paragraph
  • <h1> to <h6> â€“ Headings
  • <ul>, <ol> â€“ Lists

2. Inline Elements

Inline elements do not begin on a new line. They occupy only as much width as necessary. Common inline elements include:

  • <span> â€“ Inline container for styling
  • <a> â€“ Creates a hyperlink
  • <strong> â€“ Bold text
  • <em> â€“ Italic text

3. Empty Elements

Empty elements do not have any content or closing tags. They are self-contained. Examples include:

  • <br> â€“ Line break
  • <hr> â€“ Horizontal rule
  • <img> â€“ Image element

Complete List of HTML Tags

<html>

The <html> tag is the root element of an HTML document. It wraps all other HTML elements. It defines the document type and language attributes which help browsers and search engines. All metadata, styles, scripts, and visible content are enclosed within the <html> tag. It ensures proper rendering and interpretation of the HTML document. Including the lang attribute improves accessibility and SEO. 

<html lang="en">
    <head></head>
    <body></body>
  </html>

<head>

The <head> tag contains metadata about the HTML document. It includes information like the document title, character encoding, viewport settings, and links to external resources. This section is not displayed on the webpage, but it is essential for SEO and proper rendering. Styles and scripts can be linked or embedded inside the head. Using the head tag correctly helps in organizing document information. 

<head>
    <title>My Page</title>
    <meta charset="UTF-8">
  </head>

<body>

The <body> tag contains all the visible content of an HTML document. Everything you see on a webpageâ€”text, images, videos, forms, etc.â€”is placed inside the body. This section is rendered by the browser and is the primary area for user interaction. Well-structured body content improves accessibility and SEO. JavaScript can be placed here to enhance interactivity. 

<body>
    <p>Hello, World!</p>
  </body>

<h1> to <h6>

Heading tags (<h1> to <h6>) define the structure and hierarchy of content. <h1> is the largest and most important heading; subsequent headings decrease in size. They are crucial for both user readability and SEO, as search engines use headings to understand content structure. Proper use of headings helps in organizing the document logically. It is recommended to use only one <h1> per page. 

<h1>Main Heading</h1>
  <h2>Subheading</h2>

<p>

The <p> tag is used to define paragraphs in HTML. It automatically creates a block of text with spacing before and after. This tag is essential for breaking up large amounts of text into readable sections. You can style paragraphs using CSS to adjust spacing, alignment, and font properties. It is one of the most commonly used elements in HTML documents. 

<p>This is a paragraph.</p>

<a>

The <a> tag creates hyperlinks, allowing users to navigate between web pages. It requires the href attribute to specify the destination URL. The target attribute (like target="_blank") can be used to open links in a new tab. This tag can wrap text, images, or other elements. Using descriptive link text improves accessibility and SEO. 

<a href="https://www.example.com" target="_blank">Visit Example</a>

<img>

The <img> tag is used to display images on a webpage. It requires the src attribute to specify the image URL and the alt attribute for accessibility. Alt text helps search engines understand the image and aids screen readers. You can control image dimensions with width and height attributes. Styling can further adjust the imageâ€™s appearance using CSS. 

<img src="https://via.placeholder.com/150" alt="Sample Image" width="150">

<div>

The <div> tag is a block-level container used to group other elements. It does not inherently change the presentation of content, but it is widely used for layout and styling. Developers use divs to structure a page into sections. With CSS, divs can be styled, positioned, and manipulated. They help in organizing complex web pages. 

<div>This is a container</div>

<span>

The <span> tag is an inline container used to group text or elements for styling purposes. Unlike <div>, it does not start on a new line. It is ideal for styling small portions of text within a block. Using CSS, you can change the color, font, and other properties of the span. It is useful for applying specific formatting to parts of a sentence. 

<span style="color: red;">This text is red</span>

<ul>, <ol>, <li>

The <ul> tag creates an unordered list with bullet points. The <ol> tag creates an ordered list with numbers. Each list item is defined by the <li> tag. Lists help structure content in a clear and organized manner. They are especially useful for navigation menus, instructions, or grouped data. 

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
  </ul>

<form>

The <form> tag is used to define an HTML form for user input. It contains various input elements like text fields, checkboxes, radio buttons, and buttons. The action attribute specifies where to send the form data, and the method attribute defines how to send it (GET or POST). Forms are essential for submitting user data, such as login forms, surveys, and sign-up forms. 

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
  </form>

<input>

The <input> tag is used to create interactive controls in a form, such as text fields, checkboxes, radio buttons, and submit buttons. The type attribute specifies the type of input, such as text, password, radio, checkbox, etc. This tag is commonly used in forms to collect user input. It is one of the most versatile tags in HTML and can accept various data types based on the type attribute. 

<input type="text" id="username" name="username">

<button>

The <button> tag is used to create clickable buttons in HTML. The type attribute can be set to button, submit, or reset. The button can trigger form submissions, reset form fields, or perform other JavaScript actions when clicked. It is an essential part of user interactivity on web pages. You can style buttons using CSS to customize their appearance. 

<button type="button">Click Me</button>

<select>

The <select> tag is used to create a dropdown list. It contains <option> elements, each representing a choice in the list. The name attribute identifies the dropdown, and each option has a value that is submitted when the form is submitted. It is commonly used for selecting options like countries, payment methods, or preferences. 

<select name="cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="mercedes">Mercedes</option>
  </select>

<option>

The <option> tag is used inside a <select> tag to define the individual options in a dropdown list. Each <option> element has a value attribute, which is sent when the form is submitted. The text inside the <option> tag is the visible label in the dropdown list. 

<select>
    <option value="apple">Apple</option>
    <option value="banana">Banana</option>
  </select>

<textarea>

The <textarea> tag is used to create a multi-line text input field, allowing users to enter large amounts of text. It is commonly used for comments, feedback, or any other area requiring more than a single line of text. The rows and cols attributes define the visible size of the textarea. You can also set a default value inside the textarea. 

<textarea rows="4" cols="50">Default text</textarea>

<table>

The <table> tag is used to create a table structure in HTML. It is often used for displaying tabular data. Tables are organized into rows using the <tr> tag, and columns are defined with <td> (table data) or <th> (table header). Tables can have a border using the border attribute or styled using CSS. 

<table border="1">
    <tr><th>Name</th><th>Age</th></tr>
    <tr><td>John</td><td>30</td></tr>
  </table>

<tr>

The <tr> tag is used to define a row in an HTML table. Inside a table row, you can place table headers <th> or table data <td> elements. It is an essential part of creating tabular data layouts in HTML. 

<table>
    <tr><td>Item 1</td><td>Price 1</td></tr>
    <tr><td>Item 2</td><td>Price 2</td></tr>
  </table>

<td>

The <td> tag defines a cell in a table. It is used inside a <tr> (table row) to hold data for a particular column. It is essential for populating the table with content such as text, numbers, or images. 

<table>
    <tr><td>Apple</td><td>10</td></tr>
  </table>

<th>

The <th> tag is used to define a table header cell. It differs from <td> (table data) by displaying text in bold and centered by default. Header cells are typically used at the top of each column to describe the data in that column. 

<table>
    <tr><th>Name</th><th>Age</th></tr>
    <tr><td>John</td><td>30</td></tr>
  </table>

<caption>

The <caption> tag is used to provide a title or description for a table. It is placed directly after the <table> tag. It helps provide context to the data in the table, improving accessibility and readability. 

<table>
    <caption>Student List</caption>
    <tr><th>Name</th><th>Grade</th></tr>
    <tr><td>Alice</td><td>A</td></tr>
  </table>

<div>

The <div> tag is used to group content together and apply styles to it as a block-level element. It does not have any specific meaning, but it is commonly used for layout purposes or to group related content. Divs are used frequently for creating sections of a webpage, such as headers, footers, or containers for other elements. 

<div class="container">
    <p>This is a paragraph inside a div.</p>
  </div>

<span>

The <span> tag is an inline container used to style a portion of text or other inline elements without breaking the flow of the content. It does not have any inherent styling or functionality, but it is useful for applying CSS classes or attributes to specific parts of a text. The <span> tag is commonly used within paragraphs, links, or other inline elements. 

<span class="highlight">This is a highlighted text</span>

<iframe>

The <iframe> tag is used to embed another document within the current HTML page. It is commonly used for displaying videos, maps, or other web pages. The src attribute specifies the URL of the document to be embedded, and you can also control the width and height of the iframe. 

<iframe src="https://www.example.com" width="600" height="400"></iframe>

<audio>

The <audio> tag is used to embed audio content in a web page. The controls attribute provides playback controls such as play, pause, and volume control. You can specify multiple sources for different audio formats, such as MP3, Ogg, or WAV, using the <source> tag. 

<audio controls>
    <source src="audio.mp3" type="audio/mp3">
    Your browser does not support the audio element.
  </audio>

<video>

The <video> tag is used to embed a video file on a web page. It can contain one or more <source> tags for different video formats. The controls attribute adds controls for playback, such as play, pause, and volume control. You can also specify the width and height of the video. 

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video element.
  </video>

<link>

The <link> tag is used to link external resources like stylesheets (CSS files), icons, or web fonts to an HTML document. It is typically placed inside the <head> section of the page. The rel attribute specifies the relationship between the current document and the linked resource. 

<link rel="stylesheet" href="styles.css">

<meta>

The <meta> tag provides metadata about an HTML document, such as the character set, page description, keywords, and author. It is placed in the <head> section of the HTML document and does not display any content on the page itself. The <meta> tag is commonly used for setting the character encoding (e.g., UTF-8) and defining the viewport for responsive web design. 

<meta charset="UTF-8">

<br>

The <br> tag is used to insert a line break within text. It is an empty or self-closing tag and does not require an end tag. Itâ€™s often used in situations where you want to force content onto a new line, like in poems, addresses, or when formatting text within a paragraph. 

This is a line break
Here is the new line after the break

<strong>

The <strong> tag is used to define important text. By default, text inside the <strong> tag is displayed in bold. It is semantically used to indicate that the text has strong emphasis or importance, often for accessibility purposes like screen readers. 

<strong>Important text</strong>

<em>

The <em> tag is used to emphasize text, typically displayed in italics. It indicates that the enclosed text should be emphasized or stressed, which can be useful for context or importance. 

<em>Emphasized text</em>

<code>

The <code> tag is used to define a piece of computer code. It is typically rendered in a monospace font. It is most often used to highlight code snippets within documentation, tutorials, or articles. 

<code>console.log("Hello World")</code>

<pre>

The <pre> tag is used to define preformatted text. Text inside the <pre> tag is displayed with whitespace and line breaks preserved, unlike regular HTML content where multiple spaces are collapsed into one. It is often used for displaying code, poetry, or any content where formatting matters. 

<pre>This is preformatted text. It preserves spacing and new lines.</pre>

<blockquote>

The <blockquote> tag is used to define a block of text that is a quotation from another source. The text is typically indented, and it can contain other elements such as <p> or <cite>. 

<blockquote>This is a blockquote from a famous person.</blockquote>

<cite>

The <cite> tag is used to define the title of a creative work like a book, movie, or song. It is usually displayed in italics. It is most commonly used within a <blockquote> or in reference to other cited works. 

<cite>The Great Gatsby</cite>

<mark>

The <mark> tag is used to highlight parts of text. By default, the highlighted text is displayed with a yellow background. It is often used to indicate relevant or important content, like search results or keywords in a document. 

<mark>Highlighted text</mark>

<del>

The <del> tag is used to indicate that a portion of text has been deleted or is no longer relevant. It is typically rendered with a strikethrough effect to visually indicate removal or modification of content. 

<del>This is deleted text</del>

<ins>

The <ins> tag is used to represent text that has been inserted into a document. It is typically rendered with an underline. It is often used to indicate content that has been added or updated, and is typically paired with the <del> tag to show both the deleted and inserted content. 

<ins>This is inserted text</ins>

<sub>

The <sub> tag is used to define subscript text, which is rendered slightly below the normal line of text. It is commonly used in mathematical formulas or chemical formulas. 

H<sub>2</sub>O

<sup>

The <sup> tag is used to define superscript text, which is rendered slightly above the normal line of text. It is commonly used for exponents in mathematical expressions or footnotes in text. 

X<sup>2</sup>

<iframe>

The <iframe> tag is used to embed another HTML document within the current document. It is often used for embedding external content such as videos, maps, or websites. 

<iframe src="https://www.example.com"></iframe>

<object>

The <object> tag is used to embed objects such as images, audio, videos, or other media files. It can be used for interactive content and is highly customizable. 

<object data="movie.swf"></object>

<embed>

The <embed> tag is used to embed multimedia content such as audio, video, or interactive files into a webpage. It is often used for Flash animations or embedded media. 

<embed src="animation.swf">

<audio>

The <audio> tag is used to embed audio content such as music or sound effects into a webpage. It supports different audio formats like MP3, Ogg, and WAV. 

<audio controls><source src="audio.mp3" type="audio/mp3"></audio>

<video>

The <video> tag is used to embed a video file in a webpage. It allows users to play videos directly within the browser. You can include multiple sources for compatibility across browsers. 

<video width="320" height="240" controls><source src="movie.mp4" type="video/mp4"></video>

<progress>

The <progress> tag is used to display a progress bar indicating the completion status of a task. It has attributes for value and maximum range. 

<progress value="70" max="100"></progress>

<meter>

The <meter> tag represents a scalar measurement within a known range, such as disk usage or the level of a gauge. It is often used for displaying performance or resource usage. 

<meter value="0.6"></meter>

<details>

The <details> tag is used to create a collapsible section. The <summary> tag is used to define the visible heading of the section, and the content is revealed when the section is clicked.
