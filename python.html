Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability.

  • Beginner Friendly: Easy syntax and wide usage in automation, data science, and web development.
  • Interpreted: No compilation step needed.
  • Popular: Extensive libraries and community support.

Python Syntax

Python uses indentation instead of braces to define code blocks, making the code clean and readable.

  • Case-sensitive: Variables like var and Var are different.
  • Indentation: Proper indentation is mandatory.
  • No semicolons: Statements end automatically.

Variables in Python

Variables store data and are created automatically when a value is assigned.

  • Dynamically Typed: No need to declare the type.
  • Assignment: Use = to assign values.
  • Multiple Assignments: Python allows multiple variables in one line.

Data Types in Python

Python has various built-in data types for different kinds of data.

  • Numeric: int, float, complex
  • Sequence: str, list, tuple
  • Mapping: dict
  • Boolean: True or False

Operators in Python

Operators are special symbols that perform operations on variables and values.

  • Arithmetic: +, -, *, /
  • Comparison: ==, !=, >, <
  • Logical: and, or, not

Functions in Python

Functions are reusable blocks of code used to perform a single action.

  • Definition: Use def keyword to define a function.
  • Call: Functions are called using parentheses.
  • Return: Use return to return values from a function.

Classes and Objects

Classes are blueprints for creating objects (a combination of data and functions).

  • Class: Defined using class keyword.
  • Constructor: __init__ initializes object properties.
  • Object: Created by calling the class name like a function.

Inheritance in Python

Inheritance allows one class to inherit properties and methods from another.

  • Base Class: Parent class whose features are inherited.
  • Derived Class: Child class that inherits from base class.
  • Override: Child class can override parent methods.

Polymorphism in Python

Polymorphism allows different classes to define methods with the same name but different behavior.

  • Method Overriding: Same method name in child class with new behavior.
  • Built-in Polymorphism: Functions like len() work on different data types.

Encapsulation in Python

Encapsulation hides the internal state and functionality of an object.

  • Private Attributes: Use __ prefix to declare attributes private.
  • Getter/Setter: Use methods to access private data.

Abstraction in Python

Abstraction is the concept of hiding implementation details and showing only functionality.

  • Abstract Classes: Created using the ABC module.
  • Abstract Methods: Must be implemented in subclasses.

Loops in Python

Loops are used to execute a block of code repeatedly.

  • For Loop: Iterates over a sequence like list, tuple, or string.
  • While Loop: Repeats while a condition is true.
  • Break/Continue: Control loop flow.

Conditionals in Python

Conditional statements execute different blocks of code based on conditions.

  • if: Executes code if condition is true.
  • elif: Executes if previous condition was false and this is true.
  • else: Executes if all above conditions are false.

Lists in Python

Lists are ordered, mutable collections of items.

  • Access: Use index to access elements.
  • Methods: append(), remove(), sort(), etc.
  • Mixed Data: Can hold different data types.

Tuples in Python

Tuples are ordered, immutable collections of items.

  • Immutable: Cannot change after creation.
  • Faster: More efficient than lists.
  • Access: Use index to retrieve elements.

Sets in Python

Sets are unordered collections of unique items.

  • No Duplicates: Automatically removes duplicates.
  • Set Operations: union(), intersection(), difference()
  • Unordered: No index-based access.

Dictionaries in Python

Dictionaries store data in key-value pairs.

  • Keys: Must be unique and immutable.
  • Access: Retrieve values using keys.
  • Methods: get(), items(), keys(), values()

Lambda Functions

Lambda functions are small anonymous functions defined using the lambda keyword.

  • Syntax: lambda arguments: expression
  • Use Case: Often used for short, throwaway functions.
  • Functional: Useful with map(), filter(), reduce().

Regular Expressions

Regular expressions are patterns used to match character combinations in strings.

  • Module: Use Python's built-in re module.
  • Common Functions: search(), findall(), sub()
  • Patterns: Include symbols like \d, \w, .*

File Handling

Python allows reading and writing files using built-in functions.

  • Open: Use open() with modes like 'r', 'w', 'a'.
  • Context Manager: Use with to auto-close files.
  • Read/Write: Use read(), write(), readlines().

Exception Handling

Python handles errors using try-except blocks.

  • try: Code that may raise an exception.
  • except: Code to handle the error.
  • finally: Optional block that always executes.

Modules and Packages

Modules are Python files with definitions and statements. Packages are directories of modules.

  • Import: Use import or from ... import ...
  • Standard Modules: Like math, datetime, os
  • Custom Modules: You can create your own modules.

Multithreading

Multithreading allows concurrent execution of two or more threads.

  • Thread Module: Use threading module.
  • Target Function: The function that runs in a thread.
  • start(): Begins thread execution.

GUI Programming

Python supports GUI development using libraries like Tkinter, PyQt, etc.

  • Tkinter: Standard GUI toolkit in Python.
  • Widgets: Includes buttons, labels, textboxes.
  • mainloop(): Starts the GUI event loop.
